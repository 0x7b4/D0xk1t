About

The D0x module is a comprehensive info-gathering database that enables the pentester to write "D0x", or a file that holds a collection of data of a certain target, or targets. Using this data, the tester will be able to effectively understand their target, which is a critical point in the attacker's kill chain. Usually deemed malicious and black-hat in nature, the D0x module, however, aims to help security researchers gain momentum when conducting in-the-field pentesting.
The D0x module does come with several features, improved upon based off of the prior revision. Not only does it provide an user interface for at-ease use, but also capabilities to store already-collected information, as well as import non-D0xk1t written D0x reports.
